箭头函数

使用箭头函数时需要注意以下几点：
< 函数体内的this对象，绑定定义时所在的对象，而不是使用时所在的对象（this指向window）
< 不可当作构造函数，也就是说不能使用new命令，否则会抛出错误
< 不可以使用arguments对象，因为该对象在函数体内不存在

{}的使用 箭头函数中不可加new，也就是说箭头函数不能当构造函数进行使用。
