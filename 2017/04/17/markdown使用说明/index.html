Markdown 简明语法

Markdown是一种轻量级的标记语言，语法简单，可以排出简洁的表现界面。

一，列表

  • 支持表格、代码块、LaTeX数学公式、目录
  • 可以很方便的插入到 Gmail、Evernote
  • 很容易转为 HTML 、PDF 文件
  • 可以自定义 css 文件，写出更直观优雅的笔记
    • 注：按照Markdown标准语法描述，如果你的 Markdown
      文本未能正确显示，很可能是在标记符号和文本间忘了保留一个字符的空格
